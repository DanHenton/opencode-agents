# Security Policy

## Reporting a Vulnerability

We take the security of OpenCode Agents seriously. If you believe you have found a security vulnerability, please do not open a public issue.

Instead, please report the vulnerability through GitHub's [Security Advisory](https://github.com/opencode/opencode-agents/security/advisories/new) feature. This allows for a private discussion and coordinated disclosure.

## Supported Versions

As this is a hackathon project, we currently only support the latest version on the `main` branch.

Thank you for helping us keep OpenCode Agents secure!
